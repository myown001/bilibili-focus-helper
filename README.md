## 🎯 核心功能

### ✨ 一键专注
- 🏠 **首页专注模式** - 替换推荐内容为纯净搜索界面 + 学习资源收藏管理
- 📺 **视频全屏模式** - 自动全屏，三重保护防止分心退出，支持选集切换
- 🍅 **番茄钟计时** - 内置番茄工作法计时器

### 📊 智能统计
- ⏱️ **自动记录** - 学习时长、有效时间、视频列表全自动记录
- ⭐ **质量评分** - 4维度科学评估学习质量（满分100）
- 📈 **数据可视化** - 月历视图、年度热力图、统计图表

### 💾 数据导出
- 📄 支持导出 Markdown、HTML、PNG、CSV、JSON 格式
- 📅 今日报告、周报告、月报告、自定义日期报告
- 🤔 智能反思分析和改进建议

### 🎨 贴心设计
- 🎛️ **全功能控制栏** - 全屏下可调整画质、速度、切换选集
- 🖥️ **智能宽屏模式** - 退出后仍然屏蔽推荐，可查看评论区
- 📚 **学习资源管理** - 首页收藏常用学习入口，一键直达

## 🚀 快速开始

### 1️⃣ 安装插件

1. 下载插件文件
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择插件文件夹

### 2️⃣ 首次设置（30秒）

安装后首次使用会引导你完成设置：

**步骤1：设置退出密码**
- 至少6位字符
- 用于退出全屏时验证

**步骤2：设置提醒语**
- 设置3-5条激励语句
- 每次想退出时随机显示一条

示例提醒语：
```
• "想想你的目标，不要轻易放弃"
• "再学10分钟，你会感谢现在的自己"
• "坚持就是胜利，你可以的！"
```

### 3️⃣ 开始使用

#### 🏠 首页学习
1. 打开 [bilibili.com](https://www.bilibili.com)
2. 看到纯净搜索界面
3. 点击"添加合集"保存常用学习资源（可选）
   - 支持：视频、合集、UP主、专栏等
   - 以后直接点击卡片即可进入学习
4. 搜索你要学习的内容

#### 📺 视频学习
1. 打开任意B站学习视频
2. 点击"开始专注学习"按钮
3. 自动进入全屏模式
4. **移动鼠标显示控制栏**：调整画质/速度/切换选集
5. 专心学习，数据自动记录

#### 📊 查看统计
1. 点击浏览器工具栏的插件图标
2. 查看今日学习数据和质量评分
3. 浏览月历、热力图等可视化数据

## ⌨️ 常用快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Alt + R` | 打开设置（修改密码/提醒语） |
| `Ctrl + Shift + T` | 显示/隐藏番茄钟 |
| `ESC` | 退出全屏（需验证） |

💡 **Mac用户**: 使用 `Cmd` 代替 `Ctrl`

## 💡 使用技巧

### 📌 提高学习质量的3个建议

1. **使用番茄钟**
   - 按 `Ctrl + Shift + T` 开启
   - 25分钟专注 + 5分钟休息
   - 保持良好学习节奏

2. **减少干扰**
   - 关闭无关标签页
   - 手机静音或飞行模式
   - 全屏学习，沉浸体验

3. **每日复盘**
   - 晚上查看当日数据
   - 阅读改进建议
   - 持续优化学习方法

### 📊 数据导出

1. 点击插件图标
2. 点击"导出数据"按钮
3. 选择报告类型：
   - 📅 今日详细报告（推荐每日导出）
   - 📊 周报告（推荐每周导出）
   - 📈 月报告（推荐每月导出）
4. 选择格式：Markdown / HTML / 图片
5. 自动下载

## ❓ 常见问题

<details>
<summary><b>Q1: 如何退出全屏？</b></summary>

按 `ESC` 键 → 显示提醒语 → 点击"确认退出" → 输入密码 → 退出成功

💡 临时退出：直接刷新页面（但不推荐，会影响数据记录）
</details>

<details>
<summary><b>Q2: 忘记密码怎么办？</b></summary>

方法1：按 `Ctrl + Alt + R` 重置密码  
方法2：刷新页面临时退出，然后修改密码
</details>

<details>
<summary><b>Q3: 数据会丢失吗？</b></summary>

不会！数据存储在本地，有双重备份：
- Chrome 本地存储（主要）
- localStorage 备份（副本）

建议定期导出JSON格式做云端备份
</details>

<details>
<summary><b>Q4: 为什么质量评分低？</b></summary>

常见原因：
- ❌ 暂停次数太多
- ❌ 频繁切换标签
- ❌ 连续学习时间短

查看插件中的"改进建议"获取个性化提示
</details>

<details>
<summary><b>Q5: 番茄钟在哪里？</b></summary>

按 `Ctrl + Shift + T` 显示/隐藏

如果看不到，可能在屏幕边缘，可以拖拽到可见位置
</details>

<details>
<summary><b>Q6: 如何在全屏时切换到下一集？</b></summary>

无需退出全屏！操作方法：
1. 移动鼠标到屏幕底部
2. 控制栏会自动显示
3. 点击"选集"按钮
4. 选择想看的集数
5. 视频自动切换，保持全屏

💡 适合系列课程连续学习，全程不会被推荐干扰
</details>

<details>
<summary><b>Q7: 退出全屏后为什么还是宽屏？</b></summary>

这是"智能宽屏模式"，设计特点：
- ✅ 保留评论区（方便查看学习笔记和讨论）
- ✅ 宽屏显示视频（更好的观看体验）
- ❌ 屏蔽推荐内容（继续保护专注力）
- ❌ 屏蔽广告（纯净学习环境）

**刷新页面后仍然保持宽屏模式**，右下角会显示"回到全屏学习模式"按钮

要回到完整原生界面：禁用插件 → 刷新页面即可
</details>

<details>
<summary><b>Q8: 首页的合集功能怎么用？</b></summary>

首页专注模式提供学习资源快捷入口管理：

1. **添加资源**：点击"添加合集" → 输入名称和链接
2. **一键打开**：点击卡片直接跳转到学习内容
3. **管理资源**：点击"删除"按钮移除不需要的
4. **支持类型**：视频、合集、UP主主页、专栏、分区

💡 建议收藏5-10个常用学习资源，把B站首页变成你的专属学习导航
</details>
